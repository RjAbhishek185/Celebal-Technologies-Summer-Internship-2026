# Linked Service

## Purpose

The Linked Service acts as a connection between Azure Data Factory and Azure Blob Storage.

## Configuration

| Property | Value |
|----------|-------|
| Linked Service | LS_BlobStorage |
| Service Type | Azure Blob Storage |
| Authentication | Account Key |
| Purpose | Connect Azure Data Factory to Blob Storage |

## Outcome

The Linked Service was successfully created and tested, enabling Azure Data Factory to access Blob Storage securely.

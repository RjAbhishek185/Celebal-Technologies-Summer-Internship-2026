# Mini Project

## Problem Statement

Build a complete Azure Data Factory pipeline to read a CSV file from Azure Blob Storage and copy it to another Blob Storage location after validating the metadata.

---

## Solution

The pipeline performs:

1. Metadata validation
2. File existence check
3. CSV file copy
4. Output verification

# Week 02 - SQL Analysis and Querying

## Overview

This assignment focused on designing and querying an E-Commerce relational database using MySQL. The objective was to gain practical experience with SQL for managing, retrieving, and analyzing structured data.

---

## Objectives

- Design a relational database.
- Create tables with constraints.
- Insert sample records.
- Perform CRUD operations.
- Write analytical SQL queries.
- Apply aggregate functions.
- Use joins to combine tables.

---

## Technologies Used

- MySQL
- SQL

---

## Database

**Database Name:** ShopEase

The database consists of multiple related tables representing an online shopping system.

Example tables include:

- Customers
- Products
- Orders
- Order Details
- Categories

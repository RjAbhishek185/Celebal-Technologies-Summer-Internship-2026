# Week 03 - SQL Analysis using Subqueries, CTEs and Window Functions

## Overview

This assignment focused on advanced SQL concepts used in analytical databases. The tasks involved writing complex queries using subqueries, Common Table Expressions (CTEs), and Window Functions.

---

## Objectives

- Write nested queries.
- Use Common Table Expressions.
- Apply Window Functions.
- Perform ranking and analytical operations.
- Optimize query readability.

---

## Technologies Used

- MySQL

---

## Topics Covered

### Subqueries

- Single-row Subqueries
- Multi-row Subqueries
- Correlated Subqueries

### Common Table Expressions (CTEs)

- WITH clause
- Recursive CTEs

### Window Functions

- ROW_NUMBER()
- RANK()
- DENSE_RANK()
- NTILE()
- LEAD()
- LAG()

### Aggregate Window Functions

- SUM() OVER()
- AVG() OVER()
- COUNT() OVER()

### Partitioning

- PARTITION BY
- ORDER BY

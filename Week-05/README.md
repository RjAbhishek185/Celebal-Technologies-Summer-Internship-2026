# Week 05 - Apache Spark (PySpark)

This repository contains my **Week 05 Assignment** completed during the **Celebal Technologies Summer Internship 2026**. The assignment focuses on Apache Spark (PySpark) fundamentals, including distributed data processing, data cleaning, transformations, aggregations, and building processing pipelines using a custom Retail Sales dataset.

---

## Assignment Objectives

- Understand the limitations of Hadoop MapReduce.
- Learn Spark's in-memory computing architecture.
- Perform data cleaning and preprocessing using PySpark.
- Handle duplicate and missing records.
- Apply filtering, grouping, and aggregation operations.
- Work with timestamps and data type conversion.
- Build an end-to-end Spark processing pipeline.

## Dataset

A custom **Retail Sales** dataset was created specifically for this assignment.

### Dataset Features

- 500+ transaction records
- Duplicate records
- Missing values
- Multiple stores and regions
- Multiple product categories
- Customer demographic information
- Transaction timestamps

### Dataset Columns

| Column | Description |
|---------|-------------|
| user_id | Customer identifier |
| transaction_date | Transaction date |
| transaction_id | Transaction ID |
| store_id | Store identifier |
| region | Sales region |
| city | Customer city |
| state | State |
| product_category | Product category |
| product_name | Product name |
| sale_amount | Total sale amount |
| price | Product price |
| quantity | Quantity purchased |
| status | Order status |
| age | Customer age |
| subscription | Subscription type |
| email | Customer email |
| username | Customer username |
| raw_timestamp | Transaction timestamp |
